ITS Help Desk Office 365 Microsoft Office 365 is available at no cost to all members of the ODU community in order to improve communication and collaboration across campus. Download the latest version of Office, or collaborate with others using OneDrive or OneNote. Office 365 Cybersecurity: 2FA Prompt Attacks Attackers are using phishing attacks to gain MIDAS credentials, then attempting to log in multiple times until users inadvertently allow the DUO push prompt. Once a user accepts the DUO prompt, the attacker then has access to the account for a period of time. Report suspected phishing emails to Phishing@odu.edu and any cybersecurity concerns to ITSHelp@odu.edu.
